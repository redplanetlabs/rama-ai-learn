## Principles

- Challenge readme files do not provide information that should be in the rama skill
- A challenge readme, and the test-support namespace should be minimally
  sufficient to allow the challenge to pass when the rama skills are
  use/

- information that would allow an agent to "cheat" during a challenge is
  temporarily encrypted during the running of a challenge, and is
  de-encrypted afterwards.
